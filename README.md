# AzureCloud-Manager

## Übersicht
Dies ist ein VB.NET-Tool zur Verwaltung von Azure-Ressourcen mithilfe des Azure SDK. Es ermöglicht die Authentifizierung bei Azure, das Auflisten von Ressourcengruppen und das Verwalten von Cloud-Ressourcen.

## Installation
- Installieren Sie Visual Studio mit VB.NET-Unterstützung.
- Installieren Sie die folgenden NuGet-Pakete:
  - Microsoft.Azure.Management.Fluent
  - Microsoft.Rest.ClientRuntime.Azure.Authentication

## Verwendung
Führen Sie das Tool aus, um eine umfassende Übersicht über Ihre Azure-Ressourcengruppen zu erhalten. Dieses leistungsstarke und innovative Werkzeug verbindet sich nahtlos mit Ihrem Azure-Abonnement und ermöglicht es Ihnen, alle Ressourcengruppen aufzulisten und zu verwalten. Es bietet eine benutzerfreundliche Oberfläche zur schnellen Verwaltung Ihrer Cloud-Ressourcen und hilft Ihnen dabei, Ihre Azure-Infrastruktur effizient und effektiv zu überwachen.
